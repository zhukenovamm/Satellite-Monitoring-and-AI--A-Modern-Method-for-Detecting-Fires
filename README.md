# Satellite-Monitoring-and-AI--A-Modern-Method-for-Detecting-Fires
Спутниктік мониторинг пен AI арқылы өртті анықтау — бұл ғарыштан алынған суреттерді жасанды интеллект көмегімен талдап, өрт ошақтарын ерте кезеңде анықтау әдісі. Бұл технология температураның көтерілуін, түтінді байқап, өрттің таралуын болжауға мүмкіндік береді.
